# 行善台北 (Kindness Taipei)

為 12–25 歲青少年設計的隱蔽式數位陪伴 App，外殼為「TP Bus Tracker」公車到站查詢。

## 進入方式（隱蔽）

1. **第一層**：開啟 App 看到台北公車到站清單（307、299、藍2）。
2. **第二層**：在頁面底部長按「更新時間：1分鐘前」**3 秒** → 出現計算機。
3. **第三層**：在計算機輸入 **110#** → 進入「小曉」對話空間。

## 功能

- **小曉**：發光圓球 + 文字對話（可接 Gemini API 做真實回覆）。
- **情緒河流**：頂部橫向河流，點「記錄心情」可丟入心情石。
- **SOS 協定**：對話中出現關鍵字（自殺、想死、跳樓等）→ 全螢幕顯示 1925 / 110 與當前位置。
- **呼吸練習**：關鍵字觸發或手動開啟 4-7-8 呼吸引導。
- **時光膠囊**：寫給 30 天後的自己，存於 localStorage。
- **資源地圖**：台北市友善資源列表 + 一鍵導航連結。

## 開發

```bash
npm install
npm run dev
```

設定環境變數（選填）：

- `GEMINI_API_KEY`：小曉對話使用 Gemini（在 `app/api/chat/route.ts` 使用）。
- 未設定時對話為本地模擬回覆。

## 技術

- Next.js 14、React、TypeScript
- Tailwind CSS、Framer Motion
- 關鍵字過濾與 SOS 邏輯在 `lib/sos.ts`、`lib/constants.ts`
